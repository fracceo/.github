FRACCEO (Flood Risk Assessment in a Changing Climate for Cities using EO) is an European Space Agency-funded project with the goal to to develop and demonstrate a globally transferable methodology that combines Earth Observation (EO) data with hydrodynamic modelling to improve flood risk assessments in coastal urban areas.

FRACCEO is funded by the European Space Agency's Climate Space: Climate Change and Cities Activity - EXPRO.

This repository contains Foreground IPR developed under ESA Contract No. 4000148747/25/I/LR.
It incorporates Background IPR components as defined in the contract:

- [dfm_tools](https://github.com/Deltares/dfm_tools) (GPL‑3.0, Deltares)
- [HydroMT](https://github.com/Deltares/hydromt) (MIT, Deltares)

This repository is licensed under GPL‑3.0 to ensure compatibility with dfm_tools.
